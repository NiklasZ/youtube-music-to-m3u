# YouTube Music Takeout CSV to M3U
This is a tool I wrote to convert a CSV generated by [Google's Takeout](https://takeout.google.com/) of Youtube Music to a standardised M3U playlist
that can easily be imported into other music players like Jellyfin, Navidrome, etc. This is neededed because Youtube 
by default just dumps out a CSV with an internal identifier that does not map to a song or video file. Here an example:

```
Video ID,Playlist video creation timestamp
Hseq19rJbZE,2026-02-12T10:15:52+00:00
T0rgFnbZujM,2026-02-12T10:15:52+00:00
pkSj4cWO12s,2026-02-12T10:15:52+00:00
```

This script converts this to:
```
#EXTM3U
#EXTINF:-1,Une vie à peindre
~/Desktop/youtube-music-to-m3u/data/Takeout/YouTube and YouTube Music/music (library and uploads)/Une vie à peindre.mp3
#EXTINF:-1,Paintress
~/Desktop/youtube-music-to-m3u/data/Takeout/YouTube and YouTube Music/music (library and uploads)/Paintress.mp3
#EXTINF:-1,Sirène ‐ Poème d’Amour
~/Desktop/youtube-music-to-m3u/data/Takeout/YouTube and YouTube Music/music (library and uploads)/Sirène ‐ Poème d’Amour.mp3
```

**Note**: this will only work on files you uploaded yourself (i.e your own music or videos). This script will not download 3rd party music or videos for you.

## Requirements

- Python 3.12+
- Install dependencies with `pip install -r requirements.txt`
- A Google account with access to the uploaded YouTube Music tracks

## Setting up Google Cloud
To map Google's video IDs to actual songs/videos, we need to access the Youtube Data API with user credentials:

1. Create a [GCloud project](https://cloud.google.com/console) and enable the `YouTube Data API v3`.
2. Then create an OAuth2 access key (see guide [here](https://developers.google.com/youtube/v3/docs/?apix=true)) for a `Desktop App` and save the JSON. See [api_config.example.json](api_config.example.json) for an example shape.
3. Under OAuth consent screen -> Audience -> Test Users, add your own account. 

Note: these exact steps can change over time. Check GCloud's documentation and/or ask Gemini.

## Usage
Mapping 1 CSV to 1 M3U:
```bash
python downloader.py \
    --music-dir "data/Takeout/YouTube and YouTube Music/music (library and uploads)" \
    --input "data/Takeout/YouTube and YouTube Music/playlists/Clair Obscur videos.csv" \
    --output clair_obscur_playlist.csv \
    --credentials "api_config.json"
```

Mapping a folder of CSVs to a folder of M3Us:
```bash
 python downloader.py \
    --music-dir "data/Takeout/YouTube and YouTube Music/music (library and uploads)" \
    --input-folder "data/Takeout/YouTube and YouTube Music/playlists" \
    --output-folder "data/processed" \
    --credentials "api_config.json"     
```
